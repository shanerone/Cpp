## Задание 6
*Переписать в другой файл строки, содержащие заданную комбинацию символов. Например,
строка «Сегодня старшеклассники выполняли ЕГЭ по информатике и ИКТ» содержит
комбинацию «форма».*
### Алгоритм решения
Запрашиваем ввод с клавиатуры, по отдельности записывая каждую строку в первый файл. Затем прочитываем построчно первый файл и проверяем на наличие комбинации. Если нашли - записываем во  второй файл.
#### Тесты
Ввод строк: Сегодня старшеклассники выполняли ЕГЭ по информатике и ИКТ. \
Завтра бам бам бим бим бэм бом. \
Ввод слова для поиска: формат \
Во втором файле будет: Сегодня старшеклассники выполняли ЕГЭ по информатике и ИКТ.
