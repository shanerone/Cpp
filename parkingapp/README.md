# Итоговый проект ,,Парковочная Система"

## Описание проекта

**Парковочная Система** — это настольное приложение для управления системой парковки. Приложение позволяет отслеживать припаркованные автомобили, управлять парковочными местами, вести информацию о владельцах и создавать парковочные сеансы (регистрация входа и выхода).

Проект разработан на языке **C++** с использованием фреймворка **Qt 6** (Qt Core и Qt Quick) для создания графического интерфейса.

## Цель проекта

Разработать приложение для **автоматизации управления паркингом**, которое позволяет:

- **Регистрировать парковочные места** с указанием их типа (обычные, VIP, инвалидные и т.д.)
- **Управлять информацией о владельцах** автомобилей (ФИО, телефон, город)
- **Учитывать транспортные средства** (номер машины, тип, владелец)
- **Контролировать парковочные сеансы** (время входа и выхода)
- **Отслеживать статус** каждого парковочного места (занято/свободно)
- **Получать статистику** по использованию парковки

## Архитектура и структура

### Основные компоненты

Приложение состоит из **5 основных классов**:

#### 1. **ParkingManager** (Менеджер парковки)
- Главный класс, управляющий всеми элементами системы
- Содержит списки парковочных мест, владельцев, автомобилей и сеансов парковки
- Реализует все основные операции CRUD (Create, Read, Update, Delete)
- Сигналы для обновления UI при изменении данных

#### 2. **ParkingSpot** (Парковочное место)
- Представляет одно парковочное место на стоянке
- Характеристики: номер, тип (обычное/VIP), статус (занято/свободно)
- Поддерживает изменение статуса в реальном времени

#### 3. **Owner** (Владелец)
- Хранит информацию о владельце автомобиля
- Данные: ФИО, номер телефона, город
- Связан с транспортными средствами

#### 4. **Vehicle** (Транспортное средство)
- Представляет автомобиль в системе
- Данные: номерной знак, тип автомобиля, ID владельца
- Может быть припаркован в разных сеансах

#### 5. **ParkingSession** (Сеанс парковки)
- Отслеживает каждый сеанс парковки автомобиля
- Данные: парковочное место, автомобиль, время входа, время выхода
- Может быть активным (без времени выхода) или завершённым

### Основные функции системы

**Управление парковочными местами:**
- Добавление новых парковочных мест
- Изменение статуса (занято/свободно)
- Удаление мест (при отсутствии активных сеансов)

**Управление владельцами:**
- Регистрация новых владельцев
- Обновление информации
- Удаление (при отсутствии связанных автомобилей)

**Управление транспортом:**
- Регистрация новых автомобилей
- Удаление (при отсутствии активных сеансов парковки)

**Управление парковочными сеансами:**
- Создание нового сеанса при припаркивании
- Завершение сеанса при выезде
- Удаление завершённых сеансов

**Статистика:**
- Расчёт общего количества парковочных мест
- Расчёт количества свободных мест
- Расчёт количества занятых мест

## Пример использования

При запуске приложение автоматически инициализирует систему с примерными данными:

- **Владельцы:** несколько заранее добавленных владельцев с контактными данными
- **Парковочные места:** места типов "обычное", "VIP", "инвалидное" и т.д.
- **Автомобили:** несколько зарегистрированных автомобилей

Пользователь может затем:
1. Добавлять новые места, владельцев и автомобили через интерфейс
2. Создавать парковочные сеансы при припаркивании
3. Завершать сеансы при выезде
4. Просматривать статистику парковки
