## Задание 1
*Есть перечень названий шоколада. Определить для каждого наименования шоколада, какие из
них нравятся всем n сладкоежкам, какие — некоторым из сладкоежек, и какие — никому из
сладкоежек.*
### Алгоритм решения
Создаем массив коллекций по количеству сладкоежек и для каждого вводим его любимые шоколады, уникальные дубликаты не добавляются. Затем вычисляются три результата: пересечение всех коллекций дает шоколады, нравящиеся всем, объединение всех коллекций дает шоколады, нравящиеся некоторым. Для шоколадов, которые нравятся никому, вводим полный список всех доступных шоколадов и из него удаляюм те, что нравятся хотя бы кому-то, оставшиеся — это шоколады, нравящиеся никому.

Сладкоежка 1 (вводите шоколады, 'готово' чтобы завершить):\
Марс\
Сникерс\

Готово\
Сладкоежка 2 (вводите шоколады, 'готово' чтобы завершить):\
Марс\

Готово\
Вывод:
Нравится ВСЕМ:\
марс сникерс\

марс сникерс твикс\

## Задание 2
*В молочных магазинах города Х продается сметана с жирностью 15, 20 и 25 процентов. В городе
X был проведен мониторинг цен на сметану. Напишите эффективную по времени работы и по
используемой памяти программу, которая будет определять для каждого вида сметаны, сколько
магазинов продают ее дешевле всего. На вход программе сначала подается число магазинов N. В
каждой из следующих N строк находится информация в следующем формате:
<Фирма><Улица><Жирность><Цена>.*
### Алгоритм решения
Создаем словарь для хранения информации о минимальной цене и количестве магазинов для каждой жирности сметаны. Для каждого магазина из входных данных читаются фирма, улица, жирность и цена. Если цена ниже текущей минимальной для этой жирности, обновляется минимум и счётчик устанавливается в 1. Если цена равна минимальной, счётчик увеличивается на 1. После обработки всех магазинов выводятся три числа через пробел — количество магазинов с минимальной ценой для каждой жирности.
#### Тесты
Введите количество магазинов: 8\
Магазин 1 (Фирма Улица Жирность Цена): Перекрёсток Короленко 25 90\
Магазин 2 (Фирма Улица Жирность Цена): Ашан Цветочная 15 65\
Магазин 3 (Фирма Улица Жирность Цена): Макси Привокзальная 20 78\
Магазин 4 (Фирма Улица Жирность Цена): Виктора Домодедово 15 65\
Магазин 5 (Фирма Улица Жирность Цена): Пушкин Проспект 20 80\
Магазин 6 (Фирма Улица Жирность Цена): Люкс Макро 25 88\
Магазин 7 (Фирма Улица Жирность Цена): Спартак Сокол 15 70\
Магазин 8 (Фирма Улица Жирность Цена): Альфа Тверская 20 78\
Вывод: 2 2 1
## Задание 2
*Трехмерная точка.
Создайте такой подвид сущности Точка из задачи 3.1.1, которая будет иметь не две, а три
координаты на плоскости: X,Y,Z, и возможность подсчета расстояния между двумя трехмерными
точками.*
### Алгоритм решения
Создаем базовый класс точки. Затем создаем класс точки трехмерной и в него наследуем от базового класса точки и добавляем третью координату Z. В классе Point3D реализован метод, который вычисляет расстояние между двумя трёхмерными точками по формуле Евклида: √((x₁-x₂)² + (y₁-y₂)² + (z₁-z₂)²).
#### Тесты
Введите первую точку (X Y Z): 0 0 0\
Введите вторую точку (X Y Z): 5 2 3\

Точка 1: (0, 0, 0)\
Точка 2: (5, 2, 3)\
Расстояние: 6.16441
